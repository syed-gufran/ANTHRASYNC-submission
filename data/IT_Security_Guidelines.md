# Acme Corp — IT Security & Compliance Guidelines

## 1. Password Policy
Passwords must be **at least 12 characters** and include uppercase, lowercase,
a number, and a symbol. Passwords must be changed every **90 days** and cannot
match the last 5 passwords used.

## 2. Multi-Factor Authentication (MFA)
MFA is **mandatory for all employees** accessing internal systems, email, and
the VPN. Hardware security keys are required for anyone with administrative
privileges.

## 3. Data Classification
Company data is classified into four levels: **Public, Internal, Confidential,
and Restricted**. Restricted data (e.g. customer PII, financial records) must be
encrypted both at rest and in transit using AES-256.

## 4. Acceptable Use
Company devices must not be used to install unauthorized software. Connecting
personal USB storage devices to company laptops is **prohibited** without IT
approval.

## 5. Incident Reporting
Any suspected security incident — phishing, lost device, or data breach — must
be reported to the security team within **1 hour** of discovery via
security@acme.example or the #security-incidents channel.

## 6. Compliance
Acme Corp is **SOC 2 Type II** and **GDPR** compliant. Data subject access
requests (DSARs) are fulfilled within **30 days** of a verified request.
